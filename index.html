Ocean Plastic Pollution

Plastic pollution threatens food safety and quality, human health, coastal tourism, and contributes to climate change. Plastic pollution in the ocean has a devastating impact on marine life and ecosystems.

The purpose of this project is to analyze data on mismanaged plastic in oceans.

We cleaned our data using Python in Jupyter Notebook, imported our cvs files to a AWS relational database. We used PostgreSQL to view our data. We used linear and logistical regression machine learning models to forecast trends and make predictions. We then created a dashboard in Tableau to display our findings.

Mapping Earthquakes

In this project we created an earthquake map with two different maps and an earthquake overlay. Next, we visualized the earthquake data in relation to the tectonic plates’ location on the earth, and show all the earthquakes with a magnitude greater than 4.5 on the map, and the data on a third map. The maps can be shown in different views: street, satelite and dark maps.

Using JavaScript, Leaflet.js, and geoJSON data, I added tectonic plate data using d3.json(), added the data using the geoJSON() layer, set the tectonic plate LineString data to stand out on the map, and added the tectonic plate data to the overlay object with the earthquake data.
